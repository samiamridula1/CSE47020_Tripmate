# Tripmate

Tripmate is a web application for planning and managing trips. Users can create trips, manage checklists, track expenses, share experiences, and collaborate with friends.

## Tech Stack
- Frontend: React (Vite)
- Backend: Express (Node.js)
- Database: MongoDB

## Getting Started
To run the project locally:

**Frontend**
```
cd vite_frontend
npm install
npm run dev
```

**Backend**
```
cd backend
npm install
npx nodemon server.js
```

## Main Features
- User registration
- Profile management
- Trip creation
- Trip template
- Trip management
- Trip report
- Trip Photo gallery
- Packing checklist
- Expense tracking
- Experience sharing
- Experience management
- Experience list with report
- Experience filter and search
- Comment on experiences
